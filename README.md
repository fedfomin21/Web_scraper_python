# Web Scraper - Веб-скраппер новостей

Инструмент для сбора новостей с различных веб-сайтов.

## Быстрый старт

### 1. Клонирование проекта

```bash
git clone https://github.com/your-username/news-scraper.git
cd news-scraper
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Запуск проекта

```bash
python main.py
```
Данные будут сохранены в папке data
